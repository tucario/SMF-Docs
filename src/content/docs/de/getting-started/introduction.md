Smarter Files ist eine Lightning Web Component für Salesforce, die die Standard Files Related List durch einen leistungsstarken, vollständig konfigurierbaren Dateimanager ersetzt.

![Smarter Files auf einer Salesforce-Datensatzseite](/docs/record-page.png)

## Warum Smarter Files?

Die Standard Salesforce Files Related List ist eingeschränkt — keine Massendownloads, keine Upload-Einschränkungen, keine Sichtbarkeitssteuerungen. Smarter Files füllt diese Lücken und bleibt dabei zu 100% nativ auf der Salesforce-Plattform.

## Editionen

| Funktion | Free Edition | AppExchange Edition |
|---|---|---|
| Listen- & Kachelansichten | Ja | Ja |
| Massen-ZIP-Download | Ja | Ja |
| Öffentliche Links | Ja | Ja |
| Upload-Einschränkungen | Ja | Ja |
| Umfangreiche Dateiaktionen | Ja | Ja |
| Benutzerdefinierte Bezeichnungen | Ja | Ja |
| Native Dateivorschau | Ja | Ja |
| Dokument-Sichtbarkeitssteuerungen | — | Ja |
| Kategorieverwaltung | — | Ja |

## Bezugsquellen

- **Free Edition** — verfügbar auf [GitHub](https://github.com/tucario/SalesforceSmarterFiles) unter der MIT-Lizenz.
- **AppExchange Edition** — demnächst verfügbar mit zusätzlichen Funktionen wie Sichtbarkeitssteuerungen und Kategorieverwaltung.

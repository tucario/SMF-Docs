## Übersicht

Generieren Sie öffentliche Freigabelinks für einzelne Dateien direkt aus der Komponente, ohne den Datensatz zu verlassen.

## So erstellen Sie einen öffentlichen Link

1. Klicken Sie auf das Aktionsmenü einer Datei.
2. Wählen Sie **Public Link**.
3. Die Komponente erstellt eine Salesforce Content Distribution und kopiert die öffentliche URL in Ihre Zwischenablage.
4. Ein Erfolgshinweis bestätigt, dass der Link erstellt wurde.

Die generierte URL ist ein Standard-Salesforce-Public-Link — sie erfordert keine Authentifizierung und kann mit jedem geteilt werden.

## Konfiguration

Öffentliche Links können pro Komponenteninstanz über die Eigenschaft **Show Public Link** im Lightning App Builder aktiviert oder deaktiviert werden.

:::note
Die Erstellung öffentlicher Links erfordert, dass Content Deliveries in Ihrer Salesforce-Org aktiviert sind. Gehen Sie zu **Setup > Content Deliveries and Public Links**, um diese Funktion zu aktivieren.
:::

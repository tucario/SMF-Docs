## Übersicht

Smarter Files ersetzt die Standard-Salesforce Files Related List durch einen vollständig konfigurierbaren Dateimanager. Es läuft als native Lightning Web Component — keine externen Dienste oder Plugins.

## Kernfunktionen

- **[Ansichten & Sortierung](/de/features/views-and-sorting/)** — Wechseln Sie zwischen Listen- und Kachelansicht, sortieren Sie nach Datum, Größe oder Name.
- **[Massen-ZIP-Download](/de/features/bulk-download/)** — Laden Sie alle Dateien als einzelnes ZIP-Archiv herunter.
- **[Dateiaktionen](/de/features/file-actions/)** — Vorschau, Download, Bearbeitung, Löschung, Freigabe und mehr über das Kontextmenü.
- **[Öffentliche Links](/de/features/public-links/)** — Generieren Sie öffentliche Freigabe-URLs für einzelne Dateien.
- **[Upload-Einschränkungen](/de/features/upload-restrictions/)** — Steuern Sie Dateitypen und -größen, die hochgeladen werden können.
- **[Benutzerdefinierte Bezeichnungen](/de/features/custom-labels/)** — Passen Sie alle benutzerorientierten Texte an und unterstützen Sie mehrere Sprachen.

## AppExchange Edition

Die AppExchange Edition bietet erweiterte Funktionen:

- **[Speichermodi](/de/features/storage-modes/)** — Standard- oder Isolated-Modus für volle Kontrolle über die Dateiverknüpfung.
- **[Sichtbarkeitssteuerungen](/de/features/visibility-controls/)** — Rollenbasierte Kategoriefilterung für Dateien.
- **[Private Dokumente](/de/features/private-documents/)** — Markieren Sie Dateien als privat für den persönlichen Gebrauch.

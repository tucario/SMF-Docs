## コンテキストメニューのアクション

コンポーネント内の各ファイルには、以下のオプションを持つアクションメニューがあります：

![ファイルアクションのコンテキストメニュー](/docs/file-actions.png)

| アクション | 説明 | 利用可能条件 |
|---|---|---|
| **View Details** | 標準のSalesforceファイル詳細ページを開く | すべてのユーザー |
| **Edit Details** | ファイル名と説明をインラインで編集 | 編集権限を持つユーザー |
| **Download** | 個別ファイルをダウンロード | すべてのユーザー |
| **Public Link** | ファイルの公開共有URLを生成 | コンポーネントプロパティで設定可能 |
| **Delete** | ファイルを完全に削除（確認あり） | 削除権限を持つユーザー |
| **Remove** | ファイルを削除せずにこのレコードから除去 | Isolatedモードのみ |
| **Set Category** | ファイルに表示カテゴリを割り当て | Isolatedモード + Manage Categories権限 |
| **Mark as Private** | ファイルを自分のみに表示 | Isolatedモードのみ |
| **Remove Private** | ファイルからプライベートフラグを解除 | Isolatedモード、ファイルオーナーのみ |

## ネイティブファイルプレビュー

ファイル名をクリックすると、標準のSalesforceファイルプレビューモーダルが開きます — 外部ビューアーやプラグインは不要です。プレビューは、Salesforceがネイティブにサポートするすべてのファイルタイプ（PDF、画像、Officeドキュメントなど）をサポートします。

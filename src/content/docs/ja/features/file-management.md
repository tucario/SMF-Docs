## 概要

Smarter Filesは、標準のSalesforceファイル関連リストを完全にカスタマイズ可能なファイルマネージャーに置き換えます。ネイティブのLightning Web Componentとして動作し、外部サービスやプラグインは不要です。

## 主要機能

- **[ビューとソート](/ja/features/views-and-sorting/)** — リストビューとタイルビューを切り替え、日付、サイズ、名前でソート。
- **[一括ZIPダウンロード](/ja/features/bulk-download/)** — すべてのファイルを1つのZIPアーカイブとしてダウンロード。
- **[ファイルアクション](/ja/features/file-actions/)** — コンテキストメニューからプレビュー、ダウンロード、編集、削除、共有など。
- **[公開リンク](/ja/features/public-links/)** — 個別ファイルの公開共有URLを生成。
- **[アップロード制限](/ja/features/upload-restrictions/)** — アップロード可能なファイルタイプとサイズを制御。
- **[カスタム表示ラベル](/ja/features/custom-labels/)** — すべてのユーザー向けテキストをカスタマイズし、複数言語をサポート。

## AppExchange Edition

AppExchange Editionでは高度な機能が追加されます：

- **[ストレージモード](/ja/features/storage-modes/)** — Standardモードまたはlsolatedモードでファイルリンクを完全制御。
- **[表示制御](/ja/features/visibility-controls/)** — ロールベースのカテゴリフィルタリング。
- **[プライベートドキュメント](/ja/features/private-documents/)** — 個人使用のためにファイルをプライベートに設定。

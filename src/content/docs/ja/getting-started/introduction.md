Smarter Filesは、Salesforceのデフォルトのファイル関連リストを強力で完全にカスタマイズ可能なファイルマネージャーに置き換えるLightning Web Componentです。

![Salesforceレコードページ上のSmarter Files](/docs/record-page.png)

## なぜSmarter Files？

標準のSalesforceファイル関連リストには制限があります — 一括ダウンロード、アップロード制限、表示制御がありません。Smarter Filesは、Salesforceプラットフォームに100%ネイティブのまま、これらのギャップを埋めます。

## 入手方法

- **Free Edition** — MITライセンスで[GitHub](https://github.com/tucario/SalesforceSmarterFiles)から利用可能。
- **AppExchange Edition** — 表示制御やカテゴリ管理などの追加機能を含め、近日公開予定。

## Overview

Smarter Files replaces the standard Salesforce Files related list with a fully configurable file manager. It runs as a native Lightning Web Component — no external services or plugins.

## Core Features

- **[Views & Sorting](/features/views-and-sorting/)** — Switch between list and tile views, sort by date, size, or name.
- **[Bulk ZIP Download](/features/bulk-download/)** — Download all files as a single ZIP archive.
- **[File Actions](/features/file-actions/)** — Preview, download, edit, delete, share, and more from the context menu.
- **[Public Links](/features/public-links/)** — Generate public sharing URLs for individual files.
- **[Upload Restrictions](/features/upload-restrictions/)** — Control file types and sizes that can be uploaded.
- **[Custom Labels](/features/custom-labels/)** — Customize all user-facing text and support multiple languages.

## AppExchange Edition

The AppExchange Edition adds advanced features:

- **[Storage Modes](/features/storage-modes/)** — Standard or Isolated mode for full control over file linking.
- **[Visibility Controls](/features/visibility-controls/)** — Role-based category filtering for files.
- **[Private Documents](/features/private-documents/)** — Mark files as private for personal use.

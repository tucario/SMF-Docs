## Overview

Every piece of text in the Smarter Files component is customizable via Salesforce Custom Labels. This allows you to:

- Translate the component into any language.
- Adjust wording to match your organization's terminology.

## Customizing Labels

1. Go to **Setup > Custom Labels** in Salesforce.
2. Find labels prefixed with `SmarterFiles_`.
3. Edit the value or add translations for other languages.

## Available Labels

Labels cover all user-facing text including:

- Button text (Download, Upload, Delete, etc.)
- Column headers
- Empty state messages
- Error messages
- Tooltips

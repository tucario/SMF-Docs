Smarter Files is a Lightning Web Component for Salesforce that replaces the default Files related list with a powerful, fully configurable file manager.

![Smarter Files on a Salesforce record page](/docs/record-page.png)

## Why Smarter Files?

The standard Salesforce Files related list is limited — no bulk downloads, no upload restrictions, no visibility controls. Smarter Files fills those gaps while staying 100% native to the Salesforce platform.

## Visão Geral

Todo texto no componente Smarter Files é personalizável via Salesforce Custom Labels. Isso permite que você:

- Traduza o componente para qualquer idioma.
- Ajuste a terminologia para corresponder à da sua organização.

## Personalizando Labels

1. Acesse **Setup > Custom Labels** no Salesforce.
2. Encontre os labels com prefixo `SmarterFiles_`.
3. Edite o valor ou adicione traduções para outros idiomas.

## Labels Disponíveis

Os labels cobrem todo o texto voltado para o usuário, incluindo:

- Texto de botões (Download, Upload, Delete, etc.)
- Cabeçalhos de colunas
- Mensagens de estado vazio
- Mensagens de erro
- Tooltips

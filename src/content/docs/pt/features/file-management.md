## Visão Geral

O Smarter Files substitui a lista relacionada padrão de Files do Salesforce por um gerenciador de arquivos totalmente configurável. Ele funciona como um Lightning Web Component nativo — sem serviços externos ou plugins.

## Funcionalidades Principais

- **[Visualizações e Ordenação](/pt/features/views-and-sorting/)** — Alterne entre visualização em lista e em blocos, ordene por data, tamanho ou nome.
- **[Download em Massa ZIP](/pt/features/bulk-download/)** — Baixe todos os arquivos como um único arquivo ZIP.
- **[Ações de Arquivo](/pt/features/file-actions/)** — Pré-visualize, baixe, edite, exclua, compartilhe e muito mais a partir do menu de contexto.
- **[Links Públicos](/pt/features/public-links/)** — Gere URLs públicas de compartilhamento para arquivos individuais.
- **[Restrições de Upload](/pt/features/upload-restrictions/)** — Controle os tipos e tamanhos de arquivo que podem ser enviados.
- **[Custom Labels](/pt/features/custom-labels/)** — Personalize todo o texto voltado para o usuário e suporte múltiplos idiomas.

## Edição AppExchange

A Edição AppExchange adiciona funcionalidades avançadas:

- **[Modos de Armazenamento](/pt/features/storage-modes/)** — Modo Standard ou Isolated para controle total sobre a vinculação de arquivos.
- **[Controles de Visibilidade](/pt/features/visibility-controls/)** — Filtragem de arquivos baseada em categorias e papéis.
- **[Documentos Privados](/pt/features/private-documents/)** — Marque arquivos como privados para uso pessoal.

Smarter Files é um Lightning Web Component para Salesforce que substitui a lista relacionada padrão de Files por um gerenciador de arquivos poderoso e totalmente configurável.

![Smarter Files em uma página de registro do Salesforce](/docs/record-page.png)

## Por Que Smarter Files?

A lista relacionada padrão de Files do Salesforce é limitada — sem downloads em massa, sem restrições de upload, sem controles de visibilidade. O Smarter Files preenche essas lacunas mantendo-se 100% nativo à plataforma Salesforce.

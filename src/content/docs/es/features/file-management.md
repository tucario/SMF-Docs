## Descripción General

Smarter Files reemplaza la lista relacionada estándar de Files de Salesforce con un gestor de archivos completamente configurable. Se ejecuta como un Lightning Web Component nativo — sin servicios externos ni complementos.

## Funciones Principales

- **[Vistas y Ordenamiento](/es/features/views-and-sorting/)** — Alterna entre vistas de lista y mosaico, ordena por fecha, tamaño o nombre.
- **[Descarga Masiva en ZIP](/es/features/bulk-download/)** — Descarga todos los archivos como un único archivo ZIP.
- **[Acciones de Archivo](/es/features/file-actions/)** — Vista previa, descarga, edición, eliminación, compartir y más desde el menú contextual.
- **[Enlaces Públicos](/es/features/public-links/)** — Genera URLs públicas para compartir archivos individuales.
- **[Restricciones de Carga](/es/features/upload-restrictions/)** — Controla los tipos y tamaños de archivo que se pueden cargar.
- **[Etiquetas Personalizadas](/es/features/custom-labels/)** — Personaliza todo el texto visible para el usuario y admite múltiples idiomas.

## Edición AppExchange

La Edición AppExchange agrega funciones avanzadas:

- **[Modos de Almacenamiento](/es/features/storage-modes/)** — Modo Standard o Isolated para control total sobre la vinculación de archivos.
- **[Controles de Visibilidad](/es/features/visibility-controls/)** — Filtrado de archivos basado en categorías y roles.
- **[Documentos Privados](/es/features/private-documents/)** — Marca archivos como privados para uso personal.

## Descripción General

Cada texto en el componente Smarter Files es personalizable mediante Custom Labels de Salesforce. Esto te permite:

- Traducir el componente a cualquier idioma.
- Ajustar la terminología para que coincida con la de tu organización.

## Personalizar Etiquetas

1. Ve a **Setup > Custom Labels** en Salesforce.
2. Busca las etiquetas con prefijo `SmarterFiles_`.
3. Edita el valor o agrega traducciones para otros idiomas.

## Etiquetas Disponibles

Las etiquetas cubren todo el texto visible para el usuario, incluyendo:

- Texto de botones (Download, Upload, Delete, etc.)
- Encabezados de columnas
- Mensajes de estado vacío
- Mensajes de error
- Tooltips

Smarter Files est un Lightning Web Component pour Salesforce qui remplace la liste associée Fichiers par défaut par un gestionnaire de fichiers puissant et entièrement configurable.
![Smarter Files sur une page d'enregistrement Salesforce](/docs/record-page.png)
## Pourquoi Smarter Files ?
La liste associée Fichiers standard de Salesforce est limitée — pas de téléchargements groupés, pas de restrictions de téléversement, pas de contrôles de visibilité. Smarter Files comble ces lacunes tout en restant 100 % natif à la plateforme Salesforce.

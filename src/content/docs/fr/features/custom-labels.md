description: Comment personnaliser les libellés et le texte dans Smarter Files.
---
## Aperçu
Chaque élément de texte du composant Smarter Files est personnalisable via les Custom Labels de Salesforce. Cela vous permet de :
- Traduire le composant dans n'importe quelle langue.
- Adapter la terminologie à celle de votre organisation.
## Personnalisation des libellés
1. Accédez à **Setup > Custom Labels** dans Salesforce.
2. Recherchez les libellés préfixés par `SmarterFiles_`.
3. Modifiez la valeur ou ajoutez des traductions pour d'autres langues.
## Libellés disponibles
Les libellés couvrent tous les textes visibles par l'utilisateur, notamment :
- Texte des boutons (Télécharger, Téléverser, Supprimer, etc.)
- En-têtes de colonnes
- Messages d'état vide
- Messages d'erreur
- Infobulles

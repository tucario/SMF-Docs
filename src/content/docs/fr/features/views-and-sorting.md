## Vues liste et vignettes
Basculez entre une vue liste compacte et une vue vignettes visuelle. Le composant mémorise la vue préférée de l'utilisateur.
![Vue liste](/docs/list-view.png)
![Vue vignettes](/docs/tile-view.png)
La vue par défaut peut être définie par l'administrateur dans les propriétés du composant (`Display Mode`).
## Tri
Les fichiers peuvent être triés par :
- **Date (Plus récent / Plus ancien)** — par date de dernière modification
- **Taille (Plus grand / Plus petit)** — par taille de fichier
- **Nom (A–Z / Z–A)** — par ordre alphabétique
L'ordre de tri par défaut est configurable via la propriété `Default Sort Order` dans le Lightning App Builder. Les utilisateurs peuvent modifier l'ordre de tri en cours d'utilisation via le menu déroulant de tri.

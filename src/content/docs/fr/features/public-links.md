## Aperçu
Générez des liens de partage publics pour les fichiers individuels directement depuis le composant, sans quitter la page de l'enregistrement.
## Comment créer un lien public
1. Cliquez sur le menu d'actions d'un fichier.
2. Sélectionnez **Public Link**.
3. Le composant crée une Salesforce Content Distribution et copie l'URL publique dans votre presse-papiers.
4. Un message de confirmation indique que le lien a été créé.
L'URL générée est un lien public Salesforce standard — il ne nécessite pas d'authentification et peut être partagé avec n'importe qui.
## Configuration
Les liens publics peuvent être activés ou désactivés par instance du composant via la propriété **Show Public Link** dans le Lightning App Builder.
:::note
La création de liens publics nécessite que les Content Deliveries soient activées dans votre org Salesforce. Accédez à **Setup > Content Deliveries and Public Links** pour activer cette fonctionnalité.
:::

## نظرة عامة

يستبدل Smarter Files قائمة الملفات المرتبطة القياسية في Salesforce بمدير ملفات قابل للتكوين بالكامل. يعمل كمكوّن Lightning Web Component أصلي — بدون خدمات أو إضافات خارجية.

## الميزات الأساسية

- **[العروض والفرز](/ar/features/views-and-sorting/)** — التبديل بين عرض القائمة وعرض البلاط، والفرز حسب التاريخ أو الحجم أو الاسم.
- **[تحميل ZIP جماعي](/ar/features/bulk-download/)** — تحميل جميع الملفات كأرشيف ZIP واحد.
- **[إجراءات الملفات](/ar/features/file-actions/)** — معاينة وتحميل وتعديل وحذف ومشاركة والمزيد من القائمة السياقية.
- **[الروابط العامة](/ar/features/public-links/)** — إنشاء روابط مشاركة عامة للملفات الفردية.
- **[قيود الرفع](/ar/features/upload-restrictions/)** — التحكم في أنواع وأحجام الملفات المسموح برفعها.
- **[التسميات المخصصة](/ar/features/custom-labels/)** — تخصيص جميع النصوص الظاهرة للمستخدم ودعم لغات متعددة.

## إصدار AppExchange

يضيف إصدار AppExchange ميزات متقدمة:

- **[أوضاع التخزين](/ar/features/storage-modes/)** — الوضع القياسي أو المعزول للتحكم الكامل في ربط الملفات.
- **[عناصر التحكم في الرؤية](/ar/features/visibility-controls/)** — تصفية الملفات القائمة على الأدوار حسب الفئة.
- **[المستندات الخاصة](/ar/features/private-documents/)** — تعليم الملفات كخاصة للاستخدام الشخصي.

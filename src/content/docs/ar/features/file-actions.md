## إجراءات القائمة السياقية

لكل ملف في المكوّن قائمة إجراءات تحتوي على الخيارات التالية:

![قائمة إجراءات الملفات السياقية](/docs/file-actions.png)

| الإجراء | الوصف | التوفر |
|---|---|---|
| **View Details** | يفتح صفحة تفاصيل الملف القياسية في Salesforce | جميع المستخدمين |
| **Edit Details** | تعديل اسم الملف والوصف مباشرة | المستخدمون الذين لديهم صلاحية التعديل |
| **Download** | تحميل الملف الفردي | جميع المستخدمين |
| **Public Link** | إنشاء رابط مشاركة عام للملف | قابل للتكوين عبر خاصية المكوّن |
| **Delete** | حذف الملف نهائياً (مع تأكيد) | المستخدمون الذين لديهم صلاحية الحذف |
| **Remove** | إزالة الملف من هذا السجل دون حذفه | وضع Isolated فقط |
| **Set Category** | تعيين فئة رؤية للملف | وضع Isolated + صلاحية Manage Categories |
| **Mark as Private** | جعل الملف مرئياً لك فقط | وضع Isolated فقط |
| **Remove Private** | إزالة علامة الخصوصية من الملف | وضع Isolated، مالك الملف فقط |

## معاينة الملف الأصلية

يؤدي النقر على اسم الملف إلى فتح نافذة معاينة الملف القياسية في Salesforce — لا حاجة لعارضات أو إضافات خارجية. تدعم المعاينة جميع أنواع الملفات التي يدعمها Salesforce أصلاً (ملفات PDF، الصور، مستندات Office، إلخ.).

:::note
هذه الميزة متاحة في **إصدار AppExchange** فقط وتتطلب تعيين **وضع التخزين** للمكوّن على **Isolated**.
:::

## نظرة عامة

تتيح عناصر التحكم في الرؤية للمسؤولين تقييد الوصول إلى الملفات بناءً على فئات المستندات وأدوار المستخدمين. يمكن تعيين الملفات لفئات، ويمكن تقييد كل فئة لأدوار محددة — لن يرى المستخدمون خارج تلك الأدوار الملفات. بالإضافة إلى ذلك، يمكن لأي مستخدم تعليم ملف كخاص، مما يجعله مرئياً فقط لنفسه وللمشاهدين المخولين.

تتم جميع عمليات التصفية **من جانب الخادم** — لا يتم إرسال الملفات التي لا يمكن للمستخدم الوصول إليها إلى المتصفح أبداً.

## متطلبات وضع التخزين

عناصر التحكم في الرؤية متاحة فقط عندما يكون **وضع التخزين** للمكوّن مُعيّناً على **Isolated**:

| | القياسي | المعزول |
|---|---|---|
| الملفات مرئية في قائمة الملفات المرتبطة القياسية | نعم | لا |
| تعيين الفئات | غير متاح | متاح |
| التعليم كخاص | غير متاح | متاح |
| تصفية الرؤية القائمة على الأدوار | غير متاح | متاح |

في الوضع المعزول، يتم ربط الملفات بالسجل حصرياً عبر سجل ربط — لا تظهر في قائمة الملفات المرتبطة القياسية في Salesforce.

## فئات المستندات

### ما هي الفئات؟

الفئة هي تسمية تُعيَّن لملف تحدد من يمكنه رؤيته. يتم تعريف الفئات من قبل المسؤول عبر **معالج التكوين** وتُخزَّن كسجلات Custom Metadata Type.

أمثلة:
- السجلات الصحية
- التقارير المالية
- العقود القانونية
- المذكرات الداخلية

### كيف تعمل تصفية الفئات

لكل فئة قائمة **أدوار مسموح بها**. عند تعيين فئة لملف:

- إذا كان دور المستخدم في القائمة المسموحة ← الملف **مرئي**
- إذا لم يكن دور المستخدم في القائمة المسموحة ← الملف **مخفي**
- إذا لم تُعيَّن أدوار للفئة ← الملف مرئي **للجميع**
- إذا لم يكن للملف فئة ← الملف مرئي **للجميع**

### تعيين الفئات للملفات

يمكن للمستخدمين الذين يمتلكون مجموعة صلاحيات **Tucario - Manage File Categories** تعيين الفئات:

1. انقر على قائمة الإجراءات للملف.
2. اختر **Set Category**.
3. اختر فئة من المنتقي، أو اختر **No Category** لإزالة التعيين الحالي.

يتم تحديث رؤية الملف فوراً.

### إدارة الفئات

تتم إدارة الفئات من خلال **معالج التكوين** في تطبيق Smarter Files:

1. افتح تطبيق **Smarter Files by Tucario** من App Launcher.
2. انقر على **Manage Document Categories**.
3. **الخطوة 1 — أنواع المستندات**: أضف فئات بأسماء وأوصاف اختيارية.
4. **الخطوة 2 — تعيين الأدوار**: لكل فئة، حدد الأدوار التي يمكنها رؤية الملفات في تلك الفئة.
5. **الخطوة 3 — المراجعة والنشر**: راجع التكوين وانقر Deploy.

:::note
**إزالة فئة:** عند إزالة نوع مستند من المعالج والنشر، لا يتم حذف سجل Custom Metadata Type الأساسي — بل يتم **إلغاء تنشيطه** (`Is_Active = false`). لم تعد الفئات غير النشطة تظهر في منتقي الفئات أو تصفي الملفات. وذلك لأن Salesforce Metadata API لا تدعم حذف سجلات Custom Metadata Type من Apex.
:::

## المستندات الخاصة

### تعليم الملفات كخاصة

يمكن لأي مستخدم تعليم ملف كخاص في الوضع المعزول:

1. انقر على قائمة الإجراءات للملف.
2. اختر **Mark as Private**.
3. يظهر رمز قفل بجانب اسم الملف.

<video controls playsinline style="width:100%; border-radius:8px;">
  <source src="/docs/make-file-private.mp4" type="video/mp4" />
</video>

يصبح الملف مرئياً فقط لـ:
- المستخدم الذي علّمه كخاص (المالك)
- المستخدمون الذين يمتلكون مجموعة صلاحيات **Tucario - View Private Documents**

### إزالة حالة الخصوصية

يمكن لمالك الملف إزالة علامة الخصوصية:

1. انقر على قائمة الإجراءات للملف الخاص.
2. اختر **Remove Private**.
3. يعود الملف إلى قواعد الرؤية العادية.

## منطق التصفية المجتمعة

عندما يكون للملف فئة وعلامة خصوصية معاً، **يجب أن ينجح كلا التحققين** ليكون الملف مرئياً:

```
For each file:
  1. Private check:
     If file is private AND user is not the owner
     AND user lacks "View Private Documents" permission → HIDE

  2. Category check:
     If file has a category with permitted roles
     AND user's role is not in the list
     AND user lacks "Manage Categories" permission → HIDE

  3. Otherwise → SHOW
```

## مجموعات الصلاحيات

| مجموعة الصلاحيات | الغرض |
|---|---|
| **Tucario Files** | الوصول الأساسي. يمنح الوصول للتطبيق ووحدات التحكم وكائن الربط. يُعيَّن لجميع المستخدمين. |
| **Tucario - Manage File Categories** | يسمح بتعيين فئات للملفات. **يتجاوز تصفية الفئات** — المستخدمون الذين يمتلكون هذه الصلاحية يرون جميع الملفات بغض النظر عن الفئة. |
| **Tucario - View Private Documents** | يسمح بعرض الملفات المعلّمة كخاصة من قبل مستخدمين آخرين. |

## حالات الاستخدام

- **السجلات الصحية** مرئية فقط لمديري الموارد البشرية والطاقم الطبي.
- **التقارير المالية** مقيدة لفريق المالية.
- **العقود القانونية** متاحة فقط لأدوار القسم القانوني.
- **المرفقات الحساسة** المعلّمة كخاصة من قبل مستخدمين فرديين للاستخدام الشخصي.
- **مسودات المستندات** مخفية عن الأدوار الخارجية حتى تصبح جاهزة.

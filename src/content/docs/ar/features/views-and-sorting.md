## عرض القائمة والبلاط

التبديل بين عرض قائمة مضغوط وعرض بلاط مرئي. يتذكر المكوّن العرض المفضل للمستخدم.

![عرض القائمة](/docs/list-view.png)

![عرض البلاط](/docs/tile-view.png)

يمكن للمسؤول تعيين العرض الافتراضي في خصائص المكوّن (`Display Mode`).

## الفرز

يمكن فرز الملفات حسب:

- **التاريخ (الأحدث / الأقدم)** — حسب تاريخ آخر تعديل
- **الحجم (الأكبر / الأصغر)** — حسب حجم الملف
- **الاسم (أ–ي / ي–أ)** — أبجدياً

ترتيب الفرز الافتراضي قابل للتكوين عبر خاصية `Default Sort Order` في Lightning App Builder. يمكن للمستخدمين تغيير ترتيب الفرز أثناء التشغيل باستخدام قائمة الفرز المنسدلة.

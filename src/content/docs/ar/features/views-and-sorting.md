## عرض القائمة

التخطيط الافتراضي — صفوف عمودية تعرض اسم الملف وأيقونة النوع وتاريخ آخر تعديل والحجم والمالك وشارة الفئة (إن عُيِّنت) وأيقونة قفل الخصوصية للملفات الخاصة. لكل صف زر قائمة سياقية يكشف إجراءات خاصة بالملف.

<Image src={listView} alt="عرض قائمة Smarter Files مع ملفين ورابط Download All Files" />

## عرض البطاقات

شبكة بطاقات مع أيقونات نوع ملف بارزة. مفيد للسجلات التي تحتوي على صور كثيرة، أو عندما يكون الاستعراض السريع أهم من قراءة البيانات الوصفية.

<Image src={tileView} alt="عرض بطاقات Smarter Files يعرض الملفات كشبكة متجاوبة من البطاقات" />

## التبديل بين العروض

يضبط المسؤول العرض **الافتراضي** عبر خاصية التصميم **Display Mode** في App Builder (`List` أو `Tiles`). يمكن للمستخدمين التبديل بين العروض أثناء التشغيل؛ يُحفظ تفضيلهم لتلك الصفحة.

## الفرز

ستة خيارات فرز، جميعها متاحة في القائمة المنسدلة للفرز:

| الخيار | يرتب حسب |
|---|---|
| التاريخ (الأحدث أولاً) | `LastModifiedDate` تنازلياً — الافتراضي |
| التاريخ (الأقدم أولاً) | `LastModifiedDate` تصاعدياً |
| الحجم (الأكبر أولاً) | `ContentSize` تنازلياً |
| الحجم (الأصغر أولاً) | `ContentSize` تصاعدياً |
| الاسم (أ–ي) | `Title` أبجدياً |
| الاسم (ي–أ) | `Title` عكس الأبجدية |

<Image src={sortMenu} alt="القائمة المنسدلة للفرز مفتوحة وتعرض خيارات الفرز الستة" />

يضبط المسؤول الترتيب **الافتراضي** عبر خاصية التصميم **Default Sort Order**. يمكن للمستخدمين تجاوزه أثناء التشغيل — يُحفظ اختيارهم لتلك الصفحة خلال الجلسة.

## الحالة الفارغة

عندما لا يحتوي السجل على ملفات (أو لا توجد ملفات يُسمح للمستخدم برؤيتها بعد تصفية الرؤية)، يعرض المكوّن حالة فارغة أنيقة مع بقاء أداة الرفع ظاهرة.

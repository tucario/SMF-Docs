## الإضافة إلى صفحة السجل

1. افتح أي صفحة سجل في **Lightning App Builder**.
2. ابحث عن **Smarter Files** في لوحة المكوّنات.
3. اسحبه إلى المنطقة المطلوبة من تخطيط الصفحة.
4. احفظ الصفحة وقم بتنشيطها.

## خصائص المكوّن

![لوحة تكوين App Builder](/docs/storagemode-isolated.png)

الخصائص التالية متاحة في الشريط الجانبي لـ Lightning App Builder:

| الخاصية | الوصف | القيمة الافتراضية |
|---|---|---|
| **Card Title** | نص عنوان المكوّن | `Files` |
| **Storage Mode** | `Standard` — الملفات مرتبطة عبر ContentDocumentLink (مرئية في قائمة الملفات المرتبطة). `Isolated` — الملفات مرتبطة عبر سجل ربط فقط (مخفية من قائمة الملفات المرتبطة، تتيح عناصر التحكم في الرؤية). | `Standard` |
| **Display Mode** | طريقة عرض الملفات: `List` (صفوف عمودية) أو `Tiles` (شبكة بطاقات) | `List` |
| **Default Sort Order** | ترتيب الفرز الأولي لقائمة الملفات. يمكن للمستخدمين تغييره أثناء التشغيل. الخيارات: `date-newest`، `date-oldest`، `size-largest`، `size-smallest`، `name-az`، `name-za` | `date-newest` |
| **Initial Files Displayed** | الحد الأقصى لعدد الملفات المعروضة مبدئياً. عيّن إلى `0` لعرض جميع الملفات. يظهر رابط "عرض الكل" عندما يكون هناك ملفات أكثر. | `5` |
| **Allowed File Extensions** | قائمة بيضاء من الامتدادات مفصولة بفواصل (مثال: `pdf,docx,png`). يُسمح فقط برفع هذه الأنواع. اتركه فارغاً للسماح بجميع الأنواع. | جميع الأنواع |
| **Excluded File Extensions** | قائمة سوداء من الامتدادات مفصولة بفواصل (مثال: `exe,bat,sh`). يتم حظر رفع هذه الأنواع. | لا شيء |
| **Max File Size (MB)** | الحد الأقصى لحجم الملفات المرفوعة بالميغابايت. عيّن إلى `0` لعدم وجود حد. | `0` (بدون حد) |

### منطق قيود الرفع

عند تكوين كل من امتدادات الملفات **المسموحة** و**المستبعدة**، تعمل معاً:

1. **يتم التحقق من القائمة المسموحة أولاً** — إذا لم يكن امتداد الملف في القائمة المسموحة، يتم حظره.
2. **يتم التحقق من القائمة المستبعدة ثانياً** — إذا اجتاز الملف التحقق من القائمة المسموحة لكن امتداده في القائمة المستبعدة، يتم حظره أيضاً.

بمعنى آخر، يجب أن يكون الملف في القائمة المسموحة **و** ليس في القائمة المستبعدة ليتم قبوله.

:::tip
في معظم الحالات، تحتاج فقط إلى أحد الإعدادين. استخدم **المسموحة** عندما تريد تقييد الرفع لمجموعة صغيرة من الأنواع المعروفة (مثال: `pdf,docx,xlsx`). استخدم **المستبعدة** عندما تريد حظر أنواع معينة مع السماح بكل شيء آخر (مثال: `exe,bat`).
:::

## نصائح الوضع

- يعمل في منطقة المحتوى الرئيسية أو الشريط الجانبي أو المناطق بالعرض الكامل.
- يمكن وضعه على أي صفحة سجل لكائن قياسي أو مخصص.
- يدعم وضع نسخ متعددة على نفس الصفحة (مثال: تكوينات مختلفة لكل علامة تبويب).

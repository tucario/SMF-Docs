## الإضافة إلى صفحة سجل

1. افتح أي صفحة سجل في **Lightning App Builder**.
2. ابحث عن **Smarter Files** في لوحة المكوّنات ضمن *Custom — Managed*.
3. اسحبه وأفلته في المنطقة المطلوبة من تخطيط الصفحة.
4. اضبط خصائص التصميم في اللوحة الجانبية.
5. احفظ الصفحة وقم بتنشيطها.

## خصائص المكوّن

يعرض المكوّن ثماني خصائص تصميم في الشريط الجانبي لـ App Builder:

| الخاصية | الوصف | القيمة الافتراضية |
|---|---|---|
| **Card Title** | نص رأس المكوّن. يتجاوز Custom Label الخاص بـ `Tucario_Files_Card_Title` لهذا المثيل تحديداً. | `Files` |
| **Storage Mode** | `Standard` — تُربط الملفات عبر `ContentDocumentLink` (مرئية في قائمة الملفات المرتبطة القياسية). `Isolated` — تُربط الملفات عبر كائن الربط `Tucario_File_Visibility__c` حصراً (مخفية من قائمة الملفات القياسية، وتُتيح عناصر التحكم في الرؤية). | `Standard` |
| **Display Mode** | طريقة عرض الملفات: `List` (صفوف عمودية مع البيانات الوصفية) أو `Tiles` (شبكة بطاقات مع أيقونات نوع الملف). | `List` |
| **Default Sort Order** | الترتيب الأولي. يمكن للمستخدمين تغييره أثناء التشغيل. الخيارات: `date-newest`، `date-oldest`، `size-largest`، `size-smallest`، `name-az`، `name-za`. | `date-newest` |
| **Initial Files Displayed** | الحد الأقصى لعدد الملفات المعروضة في البداية. يظهر رابط "Show All" عند وجود ملفات إضافية. اضبط على `0` لعرض الكل. | `5` |
| **Allowed File Extensions** | قائمة بيضاء مفصولة بفواصل (مثل `pdf,docx,png`). لا يُسمح بتحميل سوى هذه الأنواع. اتركها فارغة للسماح بالكل. | *(فارغة — الكل مسموح)* |
| **Excluded File Extensions** | قائمة سوداء مفصولة بفواصل (مثل `exe,bat,sh`). تُحظر هذه الأنواع من الرفع. | *(فارغة — لا شيء محظور)* |
| **Max File Size (MB)** | الحد الأقصى لحجم الرفع بالميغابايت. اضبط على `0` لرفع بلا حدود. | `0` |

### منطق قيود الرفع

عند ضبط القائمتين (**Allowed** و**Excluded**) معاً، تُطبَّقان بالتسلسل:

1. **تُفحص القائمة البيضاء أولاً** — إذا لم يكن امتداد الملف في القائمة البيضاء، يُرفض.
2. **تُفحص القائمة السوداء ثانياً** — إذا اجتاز الملف القائمة البيضاء لكن امتداده مدرج في القائمة السوداء، يُرفض أيضاً.

يجب أن يكون الملف في القائمة البيضاء **وألّا** يكون في القائمة السوداء حتى يُقبل.

:::tip
في معظم الحالات تحتاج إلى إحدى الإعدادين فقط. استخدم **Allowed** عندما تريد تقييد الرفع بمجموعة صغيرة من الأنواع المعروفة (مثل `pdf,docx,xlsx`). استخدم **Excluded** عندما تريد حظر أنواع بعينها والسماح بالباقي (مثل `exe,bat`).
:::

## نصائح حول التوضع

- يعمل المكوّن في منطقة المحتوى الرئيسية أو الشريط الجانبي أو المناطق الكاملة العرض في صفحة السجل.
- يمكن وضعه على أي صفحة سجل لكائن قياسي أو مخصص.
- يُدعم وجود نسخ متعددة في نفس الصفحة — مفيد لتقسيم السجل إلى مساحات ملفات مختلفة (مثلاً: نسخة بوضع Standard للملفات العامة ونسخة بوضع Isolated للمستندات السرية). عيّن **Card Titles** مختلفة للتمييز بينها.
- المكوّن مخصص لـ **صفحات السجلات فقط**. يأتي معالج التكوين كصفحة تطبيق منفصلة يمكن الوصول إليها من تطبيق *Smarter Files by Tucario* في App Launcher.

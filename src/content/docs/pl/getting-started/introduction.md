Smarter Files to Lightning Web Component dla Salesforce, który zastępuje domyślną listę powiązaną Files potężnym, w pełni konfigurowalnym menedżerem plików.

![Smarter Files na stronie rekordu Salesforce](/docs/record-page.png)

## Dlaczego Smarter Files?

Standardowa lista powiązana Files w Salesforce jest ograniczona -- brak zbiorczego pobierania, brak ograniczeń przesyłania, brak kontroli widoczności. Smarter Files wypełnia te luki, pozostając w 100% natywnym rozwiązaniem platformy Salesforce.

## Edycje

| Funkcja | Darmowa edycja | Edycja AppExchange |
|---|---|---|
| Widoki listy i kafelków | Tak | Tak |
| Pobieranie zbiorcze ZIP | Tak | Tak |
| Linki publiczne | Tak | Tak |
| Ograniczenia przesyłania | Tak | Tak |
| Rozbudowane akcje na plikach | Tak | Tak |
| Niestandardowe etykiety | Tak | Tak |
| Natywny podgląd plików | Tak | Tak |
| Kontrola widoczności dokumentów | -- | Tak |
| Zarządzanie kategoriami | -- | Tak |

## Gdzie to zdobyć

- **Darmowa edycja** -- dostępna na [GitHub](https://github.com/tucario/SalesforceSmarterFiles) na licencji MIT.
- **Edycja AppExchange** -- wkrótce dostępna z dodatkowymi funkcjami, takimi jak kontrola widoczności i zarządzanie kategoriami.

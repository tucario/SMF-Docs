## Przegląd

Smarter Files zastępuje standardową listę powiązaną Files w Salesforce w pełni konfigurowalnym menedżerem plików. Działa jako natywny Lightning Web Component -- bez zewnętrznych usług i wtyczek.

## Podstawowe funkcje

- **[Widoki i sortowanie](/pl/features/views-and-sorting/)** -- Przełączanie między widokiem listy i kafelków, sortowanie wg daty, rozmiaru lub nazwy.
- **[Pobieranie zbiorcze ZIP](/pl/features/bulk-download/)** -- Pobieranie wszystkich plików jako pojedyncze archiwum ZIP.
- **[Akcje na plikach](/pl/features/file-actions/)** -- Podgląd, pobieranie, edycja, usuwanie, udostępnianie i więcej z menu kontekstowego.
- **[Linki publiczne](/pl/features/public-links/)** -- Generowanie publicznych adresów URL do udostępniania poszczególnych plików.
- **[Ograniczenia przesyłania](/pl/features/upload-restrictions/)** -- Kontrola typów i rozmiarów plików, które mogą być przesyłane.
- **[Niestandardowe etykiety](/pl/features/custom-labels/)** -- Dostosowanie całego tekstu widocznego dla użytkownika i obsługa wielu języków.

## Edycja AppExchange

Edycja AppExchange dodaje zaawansowane funkcje:

- **[Tryby przechowywania](/pl/features/storage-modes/)** -- Tryb Standard lub Isolated dla pełnej kontroli nad powiązaniami plików.
- **[Kontrola widoczności](/pl/features/visibility-controls/)** -- Filtrowanie plików oparte na rolach i kategoriach.
- **[Dokumenty prywatne](/pl/features/private-documents/)** -- Oznaczanie plików jako prywatne do użytku osobistego.
